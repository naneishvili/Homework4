Список команд, которые мы использовали на уроке по Git

  • git init - Создание репозитория
  • git add - Добавление файла в индекс
  • git status - Просмотр текущего положения дел
  • git commit - Добаление файла в комит
  • git branch - показывает список веток
  • git checkout - переключения между ветками
  • git merge - слияние веток
  • git reset - вернутся на какую-либо предыдущую версию
